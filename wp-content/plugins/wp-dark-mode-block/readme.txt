Add a switch button to toggle between dark mode and normal mode with the Dark Mode Switch block.

== Description ==

Use WP Dark Mode to enable a stunning dark mode theme for your WordPress website. The plugin add a switch button to toggle between darkmode and normal mode of the pages.

Works across all operating systems including Android, iOS, macOS. No settings needed.

WP Dark Mode supports all major operating systems including iOS, Android, macOS or Windows 10.

=== How to Use ===
You can display a Darkmode switch button in any page/post by using the Gutenberg Block, for your users to switch between the dark and normal mode.

For displaying the Darkmode Switch button in Gutenberg use the **(Dark Mode Switch)** Block.

== Installation ==

= Modern Way: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For "Dark Mode".
3. Install in 1-click

= Old Way: =
1. Upload `Dark Mode Block` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
